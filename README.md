# Triangle Calculator & Visualizer

Консольное приложение на C# для вычисления типа треугольника по трем сторонам и генерации координат его вершин для отрисовки в области 100x100 px.

##  Описание задачи
Программа принимает на вход три строки (длины сторон A, B и C) и выполняет:
1. **Валидацию данных**: Проверка на вещественные положительные числа типа `float`.
2. **Анализ геометрии**: Определение типа (равносторонний, равнобедренный, разносторонний или не треугольник).
3. **Расчет координат**: Генерация трех пар координат (X, Y), масштабированных под область 100x100 пикселей.
4. **Логирование**: Запись всех операций в консоль и файл `log.txt`.

##  Функционал
- **Обработка исключений**: Программа не прерывается при вводе некорректных данных.
- **Масштабирование**: Автоматическое вписывание треугольника в заданный квадрат.
- **Специфические коды возврата**:
  - `(-1, -1)` — если стороны числовые, но треугольник построить нельзя.
  - `(-2, -2)` — если введены невалидные (нечисловые) данные.
